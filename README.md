# Excel a GPX

Aplicacion React para convertir un archivo Excel con coordenadas UTM Zona 19 Sur
(`EPSG:32719`) a un archivo GPX con coordenadas WGS84 (`EPSG:4326`).

## Columnas requeridas

- `Position X`
- `Position Y`
- `NAME1`

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Desplegar en Vercel

1. Sube este proyecto a GitHub.
2. Importa el repositorio en Vercel.
3. Usa el comando de build `npm run build`.
4. Publica la carpeta `dist`.
